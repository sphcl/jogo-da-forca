# Jogo da Forca
### Este repositório contém um jogo da forca desenvolvido em Python. O objetivo do jogo é adivinhar a palavra secreta escolhendo letras dentro de um número limitado de tentativas.


#### Funcionalidades:
#####   •Interface simples: Interface intuitiva com regras pré-estabelecidas.
#####   •Palavras sorteadas: A palavra é sorteada aleatoriamente de uma lista predefinida dentro do tema "países".
#####   •Feedback imediato: Indicação visual das letras corretas e incorretas.
#####   •Contador de tentativas: Exibição do número de tentativas restantes para o jogador.
#####   •Ilustrações: Ilustrações autorais alinhadas ao tema do jogo.

#### Linguagens utilizada:
##### •Python 3.12.5

#### Biblioteca utilizada:
##### •Tkinter
##### •Random
##### •Time
